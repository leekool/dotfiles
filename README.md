# dotfiles

## details

| component | tool |
|-----------|------|
| wm | hyprland |
| bar | quickshell |
| terminal | ghostty |
| editor | neovim |
| shell | zsh |
| prompt | starship |
| browser | librewolf |
| launcher | fuzzel |
| wallpaper | awww |

## structure

```
.config/
├── hypr/          # hyprland wm config
├── quickshell/    # status bar config
├── ghostty/       # terminal config
├── nvim/          # neovim config
├── librewolf/     # userChrome.css
└── starship.toml  # shell prompt
.local/
└── bin/           # scripts
.zshrc             # shell config
```
